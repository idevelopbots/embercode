Terms of Service

Last updated: July 22, 2026

Welcome to Mochi. These terms cover the basics of adding Mochi to your Discord server so both of us know where we stand.

1. Using Mochi

You need to meet Discord's own age requirement to use Mochi. Server owners and admins are responsible for how the bot is configured and used within their server.

2. What Mochi does

Mochi is a moderation and utility bot. It is not a substitute for Discord's own Trust & Safety tools, and it doesn't replace human judgment for serious moderation decisions.

3. Server data and commands

When a command is run, Mochi processes the relevant message, user, or channel data needed to carry it out — for example, logging a moderation action or a message edit.

4. Acceptable use

  • Don't use Mochi to harass, threaten, or impersonate anyone.
  • Don't try to reverse-engineer, abuse, or overload the bot's systems.
  • Don't use the bot for anything illegal where you live.

5. Changes and removal

Server admins can remove Mochi at any time from Server Settings → Integrations. We'll post in our support server if we make material changes to these terms before they take effect.

Privacy Policy

Last updated: July 22, 2026

This explains what we collect, why, and the choices you have.

1. What we collect

  • Server data: IDs for servers, channels, users, and roles needed to run commands and moderation features.
  • Moderation logs: warnings, mutes, bans, and message edits/deletions, kept per-server for auditing — configurable by admins.
  • Usage data: basic diagnostics, like crash reports and command errors.

2. How we use it

To run commands, keep moderation logs, and fix things when they break. We don't sell your data to advertisers, ever.

3. Message content

Mochi only reads message content needed to run its features (like logging edits or scanning for filtered words). Message content isn't stored beyond what's needed for logging, and it's never used to train models.

4. Your controls

Server admins can configure or disable logging features, and can request full deletion of their server's data by removing the bot or contacting us.

5. Retention

Moderation logs are kept until an admin clears them or removes the bot. Diagnostic data is kept for 90 days.
