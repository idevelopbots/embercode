Terms of Service

Last updated: July 22, 2026

Welcome to Mochi. These terms cover the basics of adding Mochi to your Discord server so both of us know where we stand.

1. Using Mochi

You need to meet Discord's own age requirement to use Mochi. Server owners and admins are responsible for how the bot is configured and used within their server.

2. What Mochi does

Mochi is a moderation and utility bot. It is not a substitute for Discord's own Trust & Safety tools, and it doesn't replace human judgment for serious moderation decisions.

3. Server data and commands

When a command is run, Mochi processes the relevant message, user, or channel data needed to carry it out — for example, logging a moderation action or a message edit.

4. Acceptable use

  • Don't use Mochi to harass, threaten, or impersonate anyone.
  • Don't try to reverse-engineer, abuse, or overload the bot's systems.
  • Don't use the bot for anything illegal where you live.

5. Changes and removal

Server admins can remove Mochi at any time from Server Settings → Integrations. We'll post in our support server if we make material changes to these terms before they take effect.
